## 安装mysys2

[安装地址](https://www.msys2.org/)

要使用**ucrt64**环境

## gcc/g++

打开ucrt64的终端

```bash
pacman -S mingw-w64-ucrt-x86_64-gcc
```

## clang/clangd

```bash
pacman -S mingw-w64-ucrt-x86_64-clang mingw-w64-ucrt-x86_64-clang-tools-extra
```

## 环境变量

以Windows11为例

设置`-->`系统`-->`系统信息`-->`高级系统设置`-->`环境变量`-->`系统变量`-->`Path`-->`添加`C:\msys64\ucrt64\bin`
